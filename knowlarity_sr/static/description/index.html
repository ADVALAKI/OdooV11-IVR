IVR Configuration

Configure IVR Authentication Token and API key in API menu under the Configuration menu
 Configure your Knowlarity SR number provided by knowlarity. You can select which SR number Hold which API and Autherization key.
Now it's All done. Let's start with knowlarity super Receptionist to make calls.

Working With Super Receptionist

You can Create, Edit, Delete agents from Agent menu
Make Call from Click to Call menu
Manage Call Logs
Schedule call to make System automatically call to your desired one.
